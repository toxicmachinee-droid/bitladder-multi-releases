# BitLadder Multichain Releases

This repository contains official Windows installer releases for BitLadder Multichain.

Download the app from the latest GitHub Release:

- `BitLadder.Multichain.Setup.<version>.exe` - Windows installer.
- `SHA256SUMS.<version>.txt` - checksums for release verification.
- `latest.json` - updater metadata.

GitHub automatically adds `Source code (zip)` and `Source code (tar.gz)` to every release. Those files are generated by GitHub from this minimal repository and are not the desktop application. End users should download the installer above.

## Public release boundary

The Git tree of this repository must contain only this `README.md`. Product source code, scripts, package manifests, configuration, credentials, private documentation, and portable builds must never be committed or uploaded here. Official releases contain only the Windows installer, `latest.json`, and SHA-256 checksums.
